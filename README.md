##### (4)小tip

​		clion软件在用GDB去debug的时候，怎么去监视，堆区动态数据的值呢？在点击小乌龟debug的时候，右边variables可以点击new watch添加

​			1.把指向堆区数组的指针，强制转换成二维数组的指针，再解引用

```c
	*(int(*)[10])(myseq.data)
```

​			2.这是GDB的书籍说到的方式

```c
	*myseq.data@10
```
